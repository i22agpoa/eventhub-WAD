# EventHub – Event Management Web Application

## 📌 Project Overview

EventHub is a web application developed for the Web Application Design course.  
The goal of this project is to design and implement a responsive front-end for an event management platform.

Users can explore events, view detailed information, and interact with the platform through different sections of the website.

---

## 🎯 Milestone 1 Objectives

- Develop a front-end web application with at least 6 distinct pages
- Use HTML and CSS (with optional Bootstrap)
- Ensure a consistent layout across all pages
- Implement responsive web design
- Provide a clear and structured user interface

---

## 🎯 Milestone 2 Objectives

- Design a relational database schema with at least 6 tables
- Implement the database using Entity Framework Core (code-first approach)
- Configure and establish a connection between the application and the database
- Generate the database using migrations
- Implement a test MVC Controller with views using Entity Framework

---

## 🌐 Website Structure

The application includes the following pages:

- Home (`index.html`)
- Events (`events.html`)
- Event Details (`details.html`)
- Create Event (`create.html`)
- User Dashboard (`dashboard.html`)
- About (`about.html`)
- Contact (`contact.html`)

---
